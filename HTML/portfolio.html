Portfolio

Pier Ratz!

Pier Ratz! is a 3D arcade/simlulator currently in development. It is based loosely on my teenage years as a beach bum in Southern California. Pier Ratz! is made using the Unity game engine and C# scripting.

Educational Computer Science Game

This game is an educational trivia based game. The user tries to navigate through a maze of 16 rooms, while answering questions about computer science. The questions are loaded in through a SQLite database, which can be updated or replaced.

Craps

Classic Craps game with full bank and betting system, and includes dice rolling animations and sounds for dice, menus, and buttons. This project was made in Java using the Java Swing framework.
